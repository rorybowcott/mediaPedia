# MediaPedia

MediaPedia is a keyboard-first, Raycast-style desktop overlay for Windows + Mac that lets you type a title and get accurate film/TV metadata in under 200ms, with progressive detail loading, local caching, operators, and fallback sources when OMDb fails.

## Stack
- Tauri v1
- React + TypeScript + Vite
- Tailwind + shadcn/ui components
- SQLite (tauri-plugin-sql)

## Requirements
- Node.js LTS (18.x)
- Rust toolchain (stable)
- Xcode CLI tools on macOS

## Quick start
```bash
npm install
npm run tauri dev
```
